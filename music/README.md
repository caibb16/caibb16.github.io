# 音乐文件目录

将你的音乐文件放在这里。

## 文件格式要求

- **音频**：.mp3, .ogg, .wav, .flac
- **封面**：.jpg, .png (建议尺寸: 300x300 或 500x500)
- **歌词**：.lrc (标准LRC格式)

## 示例结构

```
static/music/
├── song1.mp3          # 音频文件
├── song1-cover.jpg    # 专辑封面
├── song1.lrc          # 歌词文件（可选）
├── song2.mp3
└── song2-cover.jpg
```

## 歌词格式示例 (.lrc)

```
[ti:歌曲名]
[ar:歌手名]
[al:专辑名]
[by:制作人]
[00:12.00]第一句歌词
[00:17.20]第二句歌词
```
